# Prospects User Guide

**Role Required**: Prospects Only, Prospects & Cases, or Admin

---

## Quick Start

1. Click **Prospects** in navigation bar
2. Select prospect type (Tax Deed, Tax Lien, Sheriff Sale, Mortgage Foreclosure)
3. Select state and optional county
4. Results show in card grid
5. Click **View Details** to open prospect

---

## Navigation

### Main Views

**Type Selection** → **State Selection** → **County (Optional)** → **Prospect List**

### Quick Menu

| View | Purpose |
|------|---------|
| Prospects | Browse all prospects by type/state/county |
| My Prospects | Show only prospects assigned to you |
| Qualified | Prospects meeting filter criteria |
| Disqualified | Prospects not meeting criteria |
| Pending | Under review prospects |

---

## Finding Prospects

### Search Criteria

**Available Filters**:
- **County** - Text search by county name (case-insensitive)
- **State** - Text search by state name or abbreviation
- **Financial Fields** - If showing criteria results
- **Status** - Qualification status

### Search Example

**Task**: Find all prospects in Miami-Dade County, Florida

1. Click **Prospects** → Type → State (Florida)
2. In filter bar: 
   - County: `Miami-Dade`
   - State: `Florida` or `FL`
3. Click **Filter**
4. Results appear in grid below

### Clear Filters

Click **Clear** button to reset all search fields and show all prospects.

## Prospect Details

### View Information

Click **View Details** on any prospect card to see:
- **Full prospect data** (name, address, reference #)
- **Financial information** (bid amount, assessed value, judgment, sale amount)
- **Auction details** (date, type, county)
- **Status history** (changes over time)
- **Notes** (research, follow-ups, communications)

### Prospect Card Shows

| Field | Information |
|-------|-------------|
| ID | System record number |
| Type | Auction type (TD, TL, SS, MF) |
| County | County and state |
| Status | Current qualification status |
| Assigned | User this prospect is assigned to |

## Managing Prospects

### Add Note

1. Open prospect detail view
2. Scroll to notes section
3. Click **Add Note** button
4. Enter note content
5. Click **Save**

**Note Types**: Research, Follow-up, Communication

### Update Research

1. Open prospect detail view
2. Find **Research** section
3. Click **Update Research**
4. Enter latest findings
5. Save changes

### Assign Prospect

1. Go to prospect list
2. Find prospect card
3. Click **Assign** (if available)
4. Select user from dropdown
5. Confirm assignment

### Change Status

1. Open prospect detail
2. Click **Update Status** button
3. Select new status:
   - **Active** - Currently working on
   - **Monitoring** - Watching for changes
   - **Follow Up** - Needs action soon
   - **Closed Won** - Converted to case/won
   - **Closed Lost** - Not moving forward

## Card Grid Display

### Understanding Cards

Each prospect card shows:
- **Header** - ID, Type badge, Status badge
- **Location** - County and state
- **Assigned To** - Current owner
- **Contract Date** - Important milestone
- **Created** - When added to system
- **Action Button** - View full details

### Card Features

**Hover Effect**: Cards lift up with shadow for better visibility

**Color Coding**:
- **Blue badge** = Tax Deed (TD)
- **Purple badge** = Tax Lien (TL)
- **Orange badge** = Sheriff Sale (SS)
- **Green badge** = Mortgage Foreclosure (MF)

### Compact Design

Cards display maximum information in minimal space:
- Column widths adjust to screen size
- Optimal for both desktop and mobile
- 2-3 columns depending on device

## Common Tasks

### Task 1: Find High-Value Prospects in Florida

1. Click **Prospects** → Type (choose one)
2. Select **Florida**
3. Optional: Type county name in search
4. Click **Filter**
5. Cards appear with financial data
6. Review and assign to team

### Task 2: Check Your Assigned Prospects

1. Click **My Prospects**
2. See only your assigned prospects
3. Filter by county/state if needed
4. Update status as you progress

### Task 3: Qualify Prospects by Criteria

1. Click **Qualified** view
2. Shows prospects meeting configured criteria
3. Use these for priority work
4. Convert ready prospects to cases

### Task 4: Add Research Notes

1. Open prospect detail
2. Scroll to notes section
3. Click **Add Note**
4. Write findings/observations
5. Save - note appears in history

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Can't find prospect | Try broader search - remove county/state filters, search again |
| No results showing | Click Clear button, refresh page |
| Prospect not updating | Refresh page or logout/login |
| Can't assign prospect | You may not have permission - contact admin |
| Notes not saving | Check internet connection and try again |

## Tips & Best Practices

✅ **Best Practices**:
- Add notes regularly with findings
- Update status as you progress
- Use filters to organize workload
- Check qualified view daily for priority prospects
- Keep research notes detailed for team reference

❌ **Avoid**:
- Leaving prospects unassigned if doing research
- Forgetting to update status when done
- Adding vague notes without context
- Ignoring disqualified prospects (may change later)
